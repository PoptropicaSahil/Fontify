# Fontify

> 🔗 Try it yourself: https://fontify.figma.site/

You know that feeling when you know Helvetica isn't right for this project, but you also can't bear to scroll through 1000+ fonts in your system? 😵‍💫

Yeah, we felt that dread too.

So my product design sister (Saloni Girhepuje) and we decided to build something different for the Figma Make hackathon - a 3D font explorer universe that would actually make sense. 🌌 We took a neural network, trained it to understand the differences between fonts, and mapped out thousands of them in a space where similar fonts cluster together.

Want something like Roboto but with more personality? Just explore its neighborhood. 
Looking for serif fonts that don't feel stuffy? There's a whole cluster for that. ✨

🔧 How we built it: 
The tech is pretty neat - we used contrastive learning with a custom autoencoder to capture the "DNA" of each font in 512 dimensions. The model learns that fonts with similar stroke weights, curves, and personalities should live close together in this visual space. 

⚠️ Reality check: 
This whole thing was built over 2 days for the hackathon.  While we're super proud of how it looks, feels, and works, it was a short sprint - so it does take about 30 seconds for all fonts to load (assuming you're not browsing from Antarctica 🐧). Not exactly production ready, but hey, the best ideas start scrappy.

We already have a ton in mind for improvements - better performance, filtering options, that font recommender system we originally planned. 🚀
But honestly? It's still fun to play with.

Welcome to the font family! 🎨
